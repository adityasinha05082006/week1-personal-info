# Personal Information Manager

## 📌 Description
This is a simple Python program that collects, stores, and displays personal information using user input and formatted output.

## 🚀 Features
- Takes user input for personal details
- Validates input (no empty values, valid age)
- Displays information in a clean formatted way
- Calculates age in months
- Uses string formatting and methods

## 🛠️ Technologies Used
- Python 3

## 📂 Project Structure
- personal_info.py → Main program file  
- test_inputs.txt → Sample inputs for testing  
- README.md → Project documentation  

## ▶️ How to Run
1. Open terminal or command prompt  
2. Navigate to project folder  
3. Run the program:

```bash
python personal_info.py
